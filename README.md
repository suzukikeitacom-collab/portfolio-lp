📊 CSV売上集計ツール（C# / WinForms）

得意先別の売上金額を自動で集計し、画面表示および CSV として出力できる
Windows デスクトップアプリです。

CSV を選択して「集計開始」ボタンを押すだけで、
面倒な売上集計作業をワンクリックで完了できます。

🚀 主な機能

CSV ファイルの読み込み

品目 × 個数 × 単価 の売上金額を計算

得意先別に自動集計

画面（DataGridView）に結果を一覧表示

合計金額は 3 桁カンマ付き で見やすく表示

集計結果を CSV としてエクスポート

小さな画面でもボタンが隠れない レスポンシブ風レイアウト対応

📂 入力する CSV の形式

以下の 5 列を持つ CSV に対応しています。

| 列名  | 内容            |
| --- | ------------- |
| 日付  | 2025/01/01 など |
| 品目  | りんご、みかん、バナナなど |
| 個数  | 数量（整数）        |
| 単価  | 1個あたりの金額      |
| 得意先 | 会社名や取引先名      |

📘 使用方法

1. アプリを起動

2. 「CSVを選択」をクリックして入力ファイルを指定

3. 「集計開始」ボタンを押す

4. 画面に集計結果が表示されます

5. 必要に応じて結果を CSV として保存できます

🧮 集計方法

売上金額 = 個数 × 単価

得意先ごとに合計し、昇順で一覧表示します。

| 得意先    | 合計金額  |
| ------ | ----- |
| 大阪フルーツ | 1,200 |
| 東京商事   | 3,100 |
| 福岡青果   | 860   |
